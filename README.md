# Optimal Plane Change Timing in Orbital Transfers

In this project, I studied how the timing of a plane change affects the total Δv required during an orbital transfer. The main question I wanted to answer was whether it is more fuel-efficient to do the plane change at periapsis or at apoapsis.

I used standard orbital mechanics equations to model the transfer and compare both cases. These included circular velocity, the vis-viva equation, Hohmann transfer burns, and the plane change Δv equation.

The results showed that doing the plane change at apoapsis always required less total Δv than doing it at periapsis. Since the spacecraft is moving slowest at apoapsis, the plane change is cheaper there. In one example, the apoapsis case required about 346 m/s less Δv than the periapsis case.

This project helped me better understand how maneuver timing affects fuel use in orbital transfers.
