# Optimal-Plane-Change-Timing-in-Orbital-Transfers
The goal of this project is to understand how the timing of a plane change—whether it’s done at periapsis or apoapsis—affects the total Δv required during an orbital transfer. 
This is useful as it can be used both in real life and space exploration games like Kerbal Space Program or SpaceFlight Simulator to conserve precious fuel.
